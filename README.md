# EnumPOC

A simple project based on the use of enums to determine today's day based on an input representing yesterday's day of the week. The application uses numbers (0-6) to represent days, where 0 is Monday and 6 is Sunday.

## Features
- Prompts the user to enter yesterday's day as a number (0-6).
- Calculates today's day by adding 1 to the input.
- Displays the corresponding day for today.
